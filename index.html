/ phone-reminders

Phone Reminder Guide

Set these exactly on iOS Reminders or Google Calendar — copy the label as your alarm name.

📱 Setup Instructions
iOS: Open Reminders app → New List named "Job Hunt" → Add each time below → Set Repeat days shown.
Android / Google Cal: Create Event → set Notification → Repeat → check specific days.
Pro tip: Name your alarm the exact action (e.g. "📧 Send 2 job apps NOW") so you know what to do the moment it rings.
